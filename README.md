## Tecnologie

- **Next.js 14** con App Router
- **TypeScript**
- **Tailwind CSS** per lo styling
- **Supabase** per il database
- **pnpm** come package manager

## Installazione

1. Installa le dipendenze:
```bash
pnpm install
```

2. Configura le variabili d'ambiente:
Crea un file `.env.local` basandoti su `.env.local.example` e inserisci le tue credenziali Supabase.

3. Avvia il server di sviluppo:
```bash
pnpm dev
```

4. Apri [http://localhost:3000](http://localhost:3000) nel browser.

## Struttura del Progetto

- `/app` - Pagine e layout Next.js
- `/components` - Componenti React riutilizzabili
- `/lib` - Utilities e configurazioni (Supabase)
- `/public` - File statici (immagini, video)

## Pagine

- **Home** (`/`) - Hero con video background e introduzione
- **Architettura** (`/architettura`) - Descrizione della struttura esagonale
- **Collezione** (`/collezione`) - Gli arredi e i pezzi d'arte
- **Le Suite** (`/suite`) - Descrizione delle camere
- **Territorio** (`/territorio`) - L'Etna e il territorio circostante
