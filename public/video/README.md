# Video Hero

Inserisci qui il video hero per la homepage.

**Nome file:** `villa-joele-hero.mp4`

**Specifiche consigliate:**
- Formato: MP4 (H.264)
- Risoluzione: 1920x1080 o superiore
- Durata: 30-60 secondi (verrà riprodotto in loop)
- Dimensione: Ottimizzato per web (max 10-15 MB)

Il video verrà riprodotto automaticamente in background nella sezione hero della homepage.
